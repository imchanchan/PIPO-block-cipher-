# 🔐 PIPO Block Cipher (ECB & CBC Mode)

이 프로젝트는 경량 블록 암호 알고리즘인 **PIPO**를 기반으로 한 **ECB (Electronic Codebook)** 및 **CBC (Cipher Block Chaining)** 모드의 암호화 및 복호화 기능을 구현한 C 프로그램입니다.

---

## 📌 개요

- **블록 크기**: 64비트 (8바이트)
- **키 크기**: 128비트 (16바이트)
- **라운드 수**: 32 + Final round
- **언어**: C
- **모드**: ECB / CBC
